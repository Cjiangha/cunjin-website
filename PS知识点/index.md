# 快捷键
* 【Ctrl】+【D】 取消选中区域
* 【Ctrl】+【I】 反选
* 【Shift】+【F6】 羽化选区
* 【Shift】+【Ctrl】+【N】 新建图层
* 【Del】删除选区内容
* 【Shift】+【ctrl】+【z】 前进一步
* 【CTRL】+【Alt】++【z】后退一步
* 【Shift】+【F5】吸管
